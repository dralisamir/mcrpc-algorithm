mCRPC Combined Algorithm (ASCO 2025)

Interactive collapsible version based on your summarized combined algorithm.

Always (applies to all mCRPC patients)
• Continue ADT indefinitely.
• Early palliative/supportive care.
• Bone protective agent if bone mets.
• Obtain germline + somatic testing (HRR, BRCA1/2, MSI/dMMR).
• Avoid PSA-only progression decisions.
• Imaging q3–6 mo (CT CAP + bone scan).
• PSMA PET only for 177Lu-PSMA eligibility.
A1 — Prior ADT Alone
BRCA1/2+ → PARPi + ARPI (olaparib + abi, niraparib + abi, talazoparib + enza)
Other HRR+ → Talazoparib + enzalutamide (conditional)
HRR− → Abi/pred OR enzalutamide OR docetaxel
Specials:
• Bone-only symptomatic → Radium-223
• MSI-H/dMMR → Pembrolizumab
• Indolent low-volume → Sipuleucel‑T
• Oligoprogression → MDT
A2 — Prior ADT + ARPI
BRCA1/2+ → Olaparib monotherapy
Other HRR+ → Olaparib monotherapy (conditional)
HRR− → Docetaxel (avoid ARPI→ARPI)
• Sip-T (selected), local MDT possible.
• Radium‑223 if bone‑only symptomatic.
A3 — Prior ADT + Docetaxel
HRR+ → PARPi + ARPI if ARPI‑naïve
HRR− → Abi/pred OR enzalutamide OR cabazitaxel
• Radium‑223 (bone only), pembrolizumab (MSI‑H).
A4 — Prior ADT + ARPI + Docetaxel (Triplet)
PSMA+ → 177Lu‑PSMA‑617 OR cabazitaxel
HRR+ → Olaparib monotherapy
• PSMA− → Cabazitaxel ± radium‑223
• MSI‑H → Pembrolizumab
B — Neuroendocrine / Small‑Cell Variant
• Cisplatin or carboplatin + etoposide (1st line)
• Carboplatin + cabazitaxel (AVPC features)
• Clinical trial preferred after platinum
• Optional: Immunotherapy + platinum maintenance, lurbinectedin, topotecan, tarlatamab
High-Yield Memory Lines
• “Know the Past → Test the Tumor → Treat the Type.”
• BRCA: ARPI‑naïve → PARPi+ARPI; post‑ARPI → PARPi alone.
• Avoid ARPI→ARPI after failure (CABAZITAXEL wins).
• Bone-only symptomatic → Radium‑223 (not with abi/pred).
• PSMA lights the Lu → 177Lu‑PSMA‑617 after ARPI + taxane.
